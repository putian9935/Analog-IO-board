Board description

The analog IO board, version 1.0, consists of

  1. Teensy 4.1 (720 MHz Cortex-M7-based MCU)
  2. Two AD5764 (4-channel 16-bit bipolar voltage output DAC)
  3. One AD7386 (4-channel SAR ADC)

Installation

To use this library, either copy the files in the src directory to your project folder (I assume you are confident about linking), or follow the method described in Install the library to Arduino IDE to install in Arduino IDE.
