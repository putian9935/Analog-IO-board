Quickstart guide

A few examples to demonstrate the use.

Analog write

The basic access to DAC is provided by analogio_write, a function that takes channel number ch (from 0 to 7) and an unsigned 16-bit DAC number num (from 0 to 65535) as input.

Note
The DAC interprets the number in the fashion of two's complement, so sending 65535 will generate a resulting voltage close to 0.

Analog read

The basic access to ADC is provided by analogio_read, a function that takes channel (either of AIN0, AIN1, BIN0, and BIN1) as input and returns an unsigned 16-bit ADC number of that channel.

Digital read and write

The digital pins are defined as DIOn where n is an integer from 1 to 8. You may use either Arduino-style pinMode, digitalReadFast, digitalWriteFast or our style digitalio_mode, digitalio_read, digitalio_write.
