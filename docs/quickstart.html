Quickstart guide

A few examples to demonstrate the use. The mnemonic of function names is

(analog|digital)io_<verb>[_both]

the verbs for

  • the analog realm are "read" and "write";
  • the digital realm are "read", "write", and "mode".

The suffix "_both" only exists in the analog realm.

Analog write

The basic access to DAC is provided by analogio_write, a function that takes channel number ch (from 0 to 7) and an unsigned 16-bit DAC number num (from 0 to 65535) as input.

Note
The DAC interprets the number in the fashion of two's complement, so sending 65535 will generate a resulting voltage close to 0.

Example:

1
9#include "analog_io.h"
10
11void setup() {
12 while(!Serial);
13 Serial.begin(9600);
14
15 // initialize the chips
16 init_chips();
17}
18
19void loop() {
20 for(int i = -32768; i < 32768; ++i) {
21 // write DAC number i to channel 0
22 analogio_write(0, i);
23 // update channel 0 and channel 4 simultaneously
24 // analogio_write_both(0, i, 4, i);
25 }
26}

Master header for analog IO library.

Analog read

The basic access to ADC is provided by analogio_read, a function that takes channel (either of AIN0, AIN1, BIN0, and BIN1) as input and returns an unsigned 16-bit ADC number of that channel.

Example:

1
9#include <tuple> // for std::tie
10#include "analog_io.h"
11
12void setup() {
13 while (!Serial)
14 ;
15 Serial.begin(9600);
16
17 // initialize the chips
18 init_chips();
19}
20
21void loop() {
22 // single read
23 Serial.printf("ain0: %u ain1: %u bin0: %u bin1: %u\n",
24 analogio_read(AIN0),
25 analogio_read(AIN1),
26 analogio_read(BIN0),
27 analogio_read(BIN1));
28
29 // pair read
30 // in case the Teensy-bundled compiler upgrades in the future,
31 // or you may build with your latest arm toolchain
32#ifdef __cpp_structured_bindings
33 auto [ain0, bin0] = analogio_read_both(CH0);
34 auto [ain1, bin1] = analogio_read_both(CH1);
35#else
36 uint16_t ain0, bin0, ain1, bin1;
37 std::tie(ain0, bin0) = analogio_read_both(CH0);
38 std::tie(ain1, bin1) = analogio_read_both(CH1);
39#endif
40 Serial.printf("ain0: %u ain1: %u bin0: %u bin1: %u\n",
41 ain0,
42 ain1,
43 bin0,
44 bin1);
45}

Digital read and write

The digital pins are defined as DIOn where n is an integer from 1 to 8. You may use either Arduino-style pinMode, digitalReadFast, digitalWriteFast or our style digitalio_mode, digitalio_read, digitalio_write.

Example:

1
9#include "analog_io.h"
10void init() {
11 init_chips();
12 digitalio_mode(DIO1, OUTPUT);
13 // by default, a digital pin is in INPUT mode
14 // digitalio_mode(DIO2, INPUT);
15}
16
17void loop() {
18 // connect DIO1 and DIO2 externally, the circuit is a ring oscillator
19 digitalio_write(DIO1, ~digitalio_read(DIO2));
20}
