Install the library to Arduino IDE
Note
The tutorial is tested in Arduino IDE v1.8.15.
  1. Go to here and download the latest version of this branch.
  2. In Arduino IDE, go to "Sketch" > "Include Library" > "ADD .ZIP Library", locate the folder just downloaded and go to "analog_io" directory before clicking "Open".
  3. If an error of "A library named analog_io already exists" is issued, go to the sketchbook location (found in "File" > "Preferences") and locate the "libraries" folder, delete the "analog_io" folder and try step 2 again.
